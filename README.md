# BDD EdR — Competitive Dashboard

Ce projet contient le dashboard React/Vite d'intelligence concurrentielle EdRAM.

## Lancement rapide

Depuis la racine du projet, lancez :

```bash
./start.sh
```

Le script `start.sh` :
- se place dans le dossier `dashboard/`
- installe les dépendances si nécessaire
- démarre le serveur de développement Vite

## Accès

Une fois lancé, ouvrez l'URL affichée dans le terminal (généralement `http://localhost:5173` ou port suivant).
